> Хронология того, **что уже сделано** — новое сверху.
> Каждая запись = один осмысленный шаг (фича, фикс, деплой, решение).
> Файл **только дописывается**. Подробности любого шага всегда можно
> увидеть в git: `git log --oneline`, `git show <hash>`.
>
> Пиши коротко: «что изменилось» и «зачем», а не «изменён файл X».

> История первого проекта на этом движке (клиент «Альфа», `ap-pdir`) — в
> [prev-project/changelog.md](./prev-project/changelog.md).

# С чего вообще начинать работу по смарту

- 1️⃣ 🚀Заводим модуль из под этапа (`Номер` и `Название`)
	- ! Номера модулей нужно продумывать заранее, чтобы сортировка по имени модуля получилась читаемой и удобной
- 2️⃣ 👩‍💼Если есть возможность, то назначаем разработчиков
- 3️⃣ Работаем дальше с двумя основными вещами
	- ⬅️➡️ Смена стадий
		- 🤖Робот по задачам
			- ✅ Закрывает задачу той стадии с которой ушли
			- ⚒️ Открывает новую задачу модуля
	- ✏️ `Ввод активности`
		- 🤖Переносит по полям
			- `Последняя активность`, `Дата активности`, `Процесс модуля` (дозапись)
			- Очищает  `Ввод активности`
- 4️⃣📅 Планирование 
	- 🅰 Вручную меняем 
		- даты `Рассмотрение`, `Разработка`
		- дни `Дни на Запуск`, `Дни на Корректировку`, `Дни на Ожидание`, `Дни на Согласование`
	- 🅱 Работаем через график в экселе
		- Сначала экспортируем нужные нам смарты в форму графика
		- Планируем в графике
		- VBA под капотом отправляет через вебхуки новые показатели из 🅰 `Вручную меняем`
	- 🤖БП на обновление
		- 🔄📅 Пересчитывает даты
		- ⚒️📅 Обновляет сроки в задачах

# LEFT_MENU: авто-бинд на cloud, вручную — на коробке

## Наблюдение

На cloud-Б24 пункт `LEFT_MENU` (пункт в левом меню портала) обычно добавляется
автоматически при установке приложения — без явного вызова
`placement.bind(LEFT_MENU, ...)`.

На коробочной версии (self-hosted Bitrix24) такого автоматизма нет —
`placement.bind` для `LEFT_MENU` нужно вызывать явно, иначе пункта меню
попросту не будет.

## Практическое следствие

`www/api/bind.php` в этом шаблоне не включает `LEFT_MENU` в список плейсментов
по умолчанию (см. массив `$placements` — актуальные плейсменты вписываются по
факту запроса пользователя, согласно `CLAUDE.md`). Если проект деплоится на
коробку клиента и приложению нужен пункт в левом меню — добавить `LEFT_MENU` в
`$placements` явно, не полагаясь на авто-бинд, который сработает только на
cloud-версии портала.
